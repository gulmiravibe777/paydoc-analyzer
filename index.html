Инструкции

1

Подготовка документов

Соберите документы по зарплате за период: табель учёта рабочего времени, служебные записки о работе в выходные/праздничные/аварийные дни, согласия работников. Документы должны быть в текстовом виде (TXT, CSV) — можно скопировать текст из Word или Excel.

2

Загрузка файлов

Перейдите на страницу «Анализатор». Перетащите файлы в зону загрузки, выберите через диалог или вставьте текст вручную. Можно загрузить до 10 файлов общим размером до 10 МБ каждый.

3

Процесс анализа

Нажмите «Начать анализ». Приложение прочитает файлы в браузере, сформирует запрос и отправит его в Claude API. Анализ занимает от 5 до 30 секунд в зависимости от объёма документов.

4

Получение результатов

Результат появится в правой панели: сводная таблица по сотрудникам, проверка полноты документов, список проблем и рекомендации, статус готовности к расчёту. Результат можно скопировать или скачать в TXT.

Пример формата документов

Табель учёта рабочего времени

=== Табель учета рабочего времени ===
Период: июль 2024
Смена: 08:00-17:00 (8 часов)

Иванов Иван Иванович
01.07 - Рабочий день (8 часов)
06.07 - Выходной (согласие: да)
Итого: 40 часов рабочих, 16 часов выходных

Служебная записка

=== Служебная записка ===
О выполнении работ в выходной день

ФИ: Иванов И.И.
Дата: 06.07.2024
Время: 08:00-17:00 (8 часов)
Вид работ: Аварийные
Согласие: получено

Согласие работника

=== Согласие работника ===
ФИ: Иванов Иван Иванович
Должность: Электромонтер

Даю согласие на:
- Работу в выходные: 06.07, 13.07 (16 часов)
- Доплату: 100% от оклада

Подпись: ___________
Дата: 01.07.2024

💡 Советы по использованию

  • Указывайте ФИО одинаково во всех документах — это упрощает сверку.
  • Загружайте документы за один и тот же период одновременно.
  • Если приложение сообщает о недостающих документах — запросите их перед расчётом зарплаты.
  • Проверяйте статус «Готовность к расчёту» перед тем, как приступать к начислениям.
  • Для больших объёмов разбивайте документы по сотрудникам или по типам.

О программе

PayDoc Analyzer — веб-приложение для менеджеров по оплате труда и HR-специалистов, которое автоматизирует анализ, проверку полноты и сверку документов по расчёту заработной платы с помощью искусственного интеллекта (Claude API от Anthropic).

Основные возможности

  • Проверка полноты документов (табель, записки, согласия)
  • Сверка данных между документами и выявление расхождений
  • Формирование сводной таблицы по сотрудникам
  • Обнаружение противоречий в датах и часах
  • Готовые рекомендации по устранению проблем
  • Экспорт результатов в TXT

FAQ

Мои данные безопасны?

Да. Документы обрабатываются в вашем браузере и отправляются только в Claude API через HTTPS. Приложение не сохраняет ваши данные на сервере — сервера с базой данных вообще нет.

Какие документы можно загружать?

TXT и CSV файлы, либо текст, скопированный из Word/Excel и вставленный вручную. Документы должны быть текстовыми — сканы и PDF без OCR не поддерживаются.

Сколько это стоит?

Сам сайт бесплатен. Вы используете собственный ключ Anthropic API, поэтому платите напрямую Anthropic по их тарифам — обычно это доли цента за один анализ.

Работает ли это без интернета?

Нет, для анализа нужен интернет — запрос отправляется в Claude API.

Где взять API-ключ?

Зарегистрируйтесь на console.anthropic.com, создайте ключ в разделе API Keys и вставьте его на странице «Анализатор».
