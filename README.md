# 📋 PayDoc Analyzer

Веб-приложение для менеджеров по оплате труда и HR-специалистов, автоматизирующее анализ, проверку полноты и сверку документов по расчёту заработной платы с помощью Claude API (Anthropic).

## Возможности

- Загрузка документов: TXT, CSV, drag & drop, вставка текста из буфера обмена
- Проверка полноты документов (табель, служебные записки, согласия)
- Сверка данных и выявление расхождений между документами
- Сводная таблица по сотрудникам (часы, доплаты, переработка)
- Рекомендации по устранению проблем и статус готовности к расчёту
- Экспорт результата в TXT, копирование в буфер обмена

## Технологии

Чистый HTML5 / CSS3 / JavaScript (без сборки и зависимостей) + Claude API (`claude-sonnet-4-6`).

## Запуск локально

Просто откройте [index.html](index.html) в браузере — сборка не требуется.

Либо разверните на GitHub Pages, Netlify, Vercel или любом статическом хостинге.

## Использование

1. Откройте страницу «Анализатор».
2. Введите свой Anthropic API-ключ (получить можно на [console.anthropic.com](https://console.anthropic.com)) — ключ хранится только в вашем браузере (`localStorage`) и отправляется напрямую в Anthropic API, минуя какой-либо сервер.
3. Загрузите документы (табель, служебные записки, согласия) — см. примеры в [example_documents.txt](example_documents.txt).
4. Нажмите «Начать анализ» и получите результат за 5–30 секунд.
5. Скачайте или скопируйте результат.

## Приватность

Приложение полностью статическое — нет бэкенда и базы данных. Документы читаются в браузере (`FileReader API`) и отправляются напрямую в Claude API по HTTPS. Ваш API-ключ никогда не передаётся третьим лицам и не покидает ваш браузер, кроме прямого запроса к `api.anthropic.com`.

## Ограничения

- Максимальный размер файла: 10 МБ
- Максимум файлов за раз: 10
- Поддерживаются только текстовые форматы (TXT, CSV)
- Требуется интернет-соединение и собственный API-ключ Anthropic
